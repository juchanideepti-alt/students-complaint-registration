Student Complaint Registration System

Project Description

The Student Complaint Registration System is a web-based application that allows students to submit complaints online and enables administrators to manage and resolve them efficiently.

Features

- Complaint Registration
- Complaint ID Generation
- Category Selection
- Priority Selection
- Complaint History
- Admin Dashboard
- Search Complaints
- Update Complaint Status
- Delete Complaints
- About Page
- Contact Page

Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap 5
- Local Storage

Project Structure

- index.html
- admin.html
- history.html
- about.html
- contact.html
- script.js
- admin.js
- history.js
- style.css

Future Enhancements

- Complaint Tracking System
